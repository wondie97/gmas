고전 Q플레이 감성의 배경 위에서 두 장의 장면을 비교해 차이를 찾고, 친구들과 실시간으로 점수를 겨뤄보세요.

플레이어 보드

플레이어 카드를 눌러 차례를 전환할 수 있어요.

    현재 장면

    -

    남은 시간 00:00
    찾은 차이 0 / 0

    새 게임을 시작해 주세요.

    하이라이트 & 로그

    발견 · 힌트 표시

      게임 방법

      1. 플레이어를 추가하고 카드를 눌러 현재 차례를 선택합니다.
      2. 새 게임을 누르면 첫 장면의 두 이미지를 볼 수 있습니다.
      3. 원본과 변형 이미지의 차이를 찾아 클릭하세요. 정답이면 빛나는 링이 표시됩니다.
      4. 힌트는 라운드당 두 번까지 사용할 수 있으며, 사용 시 소량의 점수가 차감됩니다.
      5. 제한 시간 내 모든 차이를 찾으면 자동으로 다음 장면으로 넘어갈 수 있습니다.
      ⓒ 팬메이드 · 고전 Q플레이 세계관을 재해석한 틀린그림 찾기
