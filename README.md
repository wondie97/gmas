# Q플레이 리바이벌

웹 브라우저에서 가볍게 즐길 수 있는 추억의 Q플레이 퍼즐 팬 프로젝트입니다.

## 실행 방법

별도 빌드 과정 없이 정적 파일만으로 동작합니다.

```bash
# 저장소 루트에서 간단한 정적 서버 실행 (예: 파이썬)
python -m http.server 4173
```

브라우저에서 `http://localhost:4173`으로 접속하면 됩니다.

## 게임 설명

- 2칸짜리 블록이 내려오며, 같은 색상이 세 칸 이상 인접하면 제거됩니다.
- 제거 후 위 블록이 떨어지며 연쇄가 발생하면 콤보 점수가 누적됩니다.
- 점수에 따라 레벨과 낙하 속도가 빨라집니다.
- 방향키와 스페이스바로 조작하며, 버튼으로 일시 정지/즉시 낙하가 가능합니다.

비공식 팬 제작물로, 원작 서비스와는 무관합니다.
