# Q플레이 틀린그림 찾기 아레나

브라우저에서 실행되는 팬메이드 프로젝트로, Q플레이 세계관을 재해석한 멀티플레이 틀린그림 찾기 게임입니다. 두 장의 장면을 비교해 차이를 찾고, 로컬 멀티 플레이어와 점수를 겨뤄보세요.

## 실행 방법

정적 파일로 구성되어 있으므로 별도 빌드 없이 실행할 수 있습니다.

```bash
# 저장소 루트에서 정적 서버 실행 (예: 파이썬)
python -m http.server 4173
```

브라우저에서 `http://localhost:4173`으로 접속하면 됩니다.

## 게임 특징

- **멀티플레이 지원**: 플레이어 이름을 자유롭게 추가하고 카드 클릭으로 차례를 전환합니다. 차이를 찾으면 해당 플레이어에게 점수가 부여됩니다.
- **3개의 장면**: 주말 마켓, 새벽 바닷가, Q플레이 아케이드 등 고전 감성의 배경에서 총 15개의 차이를 찾을 수 있습니다.
- **제한 시간 & 힌트**: 각 라운드마다 제한 시간이 있으며, 힌트는 라운드당 2회까지 사용 가능하지만 점수가 차감됩니다.
- **이벤트 로그**: 누가 차이를 찾았는지, 힌트를 사용했는지 로그를 통해 확인할 수 있습니다.

## 플레이 방법

1. 플레이어 이름을 입력해 추가합니다. (기본으로 1P, 2P가 준비되어 있습니다.)
2. "새 게임"을 누르면 첫 장면이 시작되고 타이머가 작동합니다.
3. 원본과 변형 이미지를 비교해 차이가 있는 지점을 클릭하세요. 정답이면 링이 표시되고 점수가 오릅니다.
4. 힌트를 사용하면 아직 찾지 못한 차이 중 하나에 깜박이는 링이 나타납니다. 힌트 사용 시 현재 플레이어의 점수가 40점 차감됩니다.
5. 제한 시간 내 모든 차이를 찾으면 라운드를 클리어하고 "다음 장면"으로 진행할 수 있습니다.

비공식 팬 프로젝트이며, 원작 Q플레이 서비스와는 무관합니다.
